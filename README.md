# NeoVim config with lazyvim

* Install neovim from https://neovim.io/

* Clear the nvim cache

```bash 
rm -rf ~/.local/share/nvim ~/.local/state/nvim ~/.cache/nvim
```

# Install

```bash
git clone https://github.com/yuxel/dotneovim ~/.config/nvim
```

Execute neovim

```bash
nvim
```
